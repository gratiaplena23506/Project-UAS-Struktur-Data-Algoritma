# Project-UAS-Struktur-Data-Algoritma
Implementasi Algoritma Dijkstra untuk penentuan rute terpendek dan tercepat pada distribusi BBM di Pontianak. Proyek UAS Struktur Data dan Algoritma (SDA) berbasis Python, NetworkX, dan Tkinter.

# Deskripsi Proyek
Proyek ini mengimplementasikan Algoritma Dijkstra untuk memberikan rekomendasi jalur distribusi yang paling efisien antara depot dan SPBU. Sistem ini memproses dua skenario optimasi:
1. Rute Terpendek: Optimasi berdasarkan jarak (meter).
2. Rute Tercepat: Optimasi berdasarkan waktu tempuh (menit) yang telah dikalkulasi dengan akumulasi hambatan lampu merah di setiap persimpangan.

# Implementasi Teknis & Struktur Data
Sesuai dengan kurikulum Struktur Data dan Algoritma, proyek ini menerapkan:
- Graf sebagai Struktur Data Utama: Menggunakan *Adjacency List* (berbasis Dictionary) untuk representasi peta jalan yang efisien.
- Optimasi Algoritma: Penggunaan *Min-Priority Queue* (via `heapq`) untuk meminimalkan kompleksitas waktu.
